# RideLink Intercom

A phone-first motorcycle intercom system using:

- Bluetooth for Rider ↔ Pillion (Prime Link)
- Wi-Fi Hotspot mesh for Rider ↔ Rider (Chain Link)
- Intelligent audio routing and future crash detection

## Current Status
UI and connectivity working.
Working on professional audio engine (jitter buffer, RTP-style framing).

Built as a personal learning project in real-time audio engineering.
